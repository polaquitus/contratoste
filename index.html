Generar HTML final con rollback AVE/Tarifarios

Un único HTML utilizable: elegís el index.html que te resolvió Claude y se descarga index_final.html ya modificado.
Qué agrega al HTML final:
  • Si se borra un AVE polinómico, elimina AVE(s), listas ajustadas y enmienda asociada, y recalcula montos.
  • Si se borra una lista ajustada / período ajustado, elimina el AVE asociado y recalcula montos.
  • Intenta engancharse automáticamente a nombres de funciones típicos de borrado (AVE y tarifarios).

Generar el HTML final

Seleccioná tu index.html resuelto y generá el archivo final.
Importante: este HTML no es un parche aparte. Su único trabajo es ensamblar un HTML final único ya fusionado, porque desde este entorno no puedo reescribir byte a byte tu fuente original local sin que me la pase el navegador.
